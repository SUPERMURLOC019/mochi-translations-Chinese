# Mochi Card 简体中文翻译 | Simplified Chinese Translation

一个为 **Mochi Card** 提供简体中文界面支持的非官方本地化项目。

An unofficial Simplified Chinese localization project for **Mochi Card**.

## 📖 简介 | Introduction

本项目旨在为中文用户提供更友好的使用体验。目前已完成约 **90%** 的界面翻译，覆盖大部分常用功能和设置页面。

This project aims to provide a better experience for Chinese users. About **90%** of the UI has been translated, covering most commonly used features and settings.

## ✨ 特性 | Features

* 🌏 简体中文本地化
  Simplified Chinese localization

* 📝 已完成约 90% 的界面翻译
  Approximately 90% of the UI has been translated

* 🔄 持续更新和完善
  Continuously maintained and improved

* 🛠 尽可能保持原版术语和表达习惯
  Preserves the original terminology and wording whenever possible

## ⚠️ 已知问题 | Known Issues

* 部分内容尚未完成翻译
  Some content is still untranslated.

* 个别术语可能存在不准确或不统一的情况
  Some terms may be inconsistent or inaccurate.

* 随着 Mochi Card 更新，可能会出现新的未翻译内容
  New untranslated content may appear after Mochi Card updates.

## 🤝 贡献 | Contributing

欢迎提交 Issue 或 Pull Request，一起完善中文翻译。

Issues and Pull Requests are welcome to help improve the translation.

## 📄 声明 | Disclaimer

本项目仅提供简体中文翻译，不包含对 Mochi Card 本体的修改。

This project only provides a Simplified Chinese translation and does not modify the Mochi Card application itself.
